## 2. Filing a Complaint / ቅሬታ ማቅረብ

### Step 1: Navigate to Complaints Page
**Screenshot 2.1: Click Complaints Button**
```
[SCREENSHOT: Homepage with Complaints button highlighted]
- Shows: User clicking on the "Complaints" button
- Action: Click the "Complaints" or "ቅሬታዎች" button
```

**Instructions:**
1. From the homepage, click on the **"Complaints"** button (or "ቅሬታዎች" in Amharic)
2. You will be taken to the Complaints page

### Step 2: Start Filing a Complaint
**Screenshot 2.2: Complaints Page**
```
[SCREENSHOT: Complaints page showing two options]
- Shows: "File Complaint" and "Follow Up" buttons
- Action: Click "File Complaint" button
```

**Instructions:**
1. On the Complaints page, you will see two options:
   - **File Complaint** (አዲስ ቅሬታ ማቅረብ)
   - **Follow Up** (ቅሬታ መከታተል)
2. Click on **"File Complaint"** to start a new complaint

### Step 3: Fill Out Section 1 - Complainant Information
**Screenshot 2.3: Section 1 - Personal Information**
```
[SCREENSHOT: Complaint form showing Section 1 fields]
- Shows: Name, Age, Gender, Submission Type fields
- Fields to fill:
  * Complainant Name (የቅሬታ ሰብ ስም)
  * Age (ዕድሜ)
  * Gender (ጾታ)
  * Submission Type: Individual or Group (የመላኪያ አይነት)
```

**Instructions:**
1. **Complainant Name** (የቅሬታ ሰብ ስም): Enter your full name
2. **Age** (ዕድሜ): Enter your age
3. **Gender** (ጾታ): Select your gender (Male/ወንድ, Female/ሴት, Other/ሌላ)
4. **Submission Type** (የመላኪያ አይነት):
   - Select **"Individual"** (ግለሰብ) if filing alone
   - Select **"Group"** (ቡድን) if filing as a group
   - If Group: Enter number of members and their information

### Step 4: Fill Out Contact Information
**Screenshot 2.4: Contact Details Section**
```
[SCREENSHOT: Form showing contact information fields]
- Shows: Phone, Email, Address fields
- Fields:
  * Phone Number (ስልክ ቁጥር)
  * Email (ኢሜይል) - Optional
  * Subcity (ክፍለ ከተማ)
  * Woreda (ወረዳ)
  * House Number (የቤት ቁጥር)
```

**Instructions:**
1. **Phone Number** (ስልክ ቁጥር): Enter your phone number (required)
2. **Email** (ኢሜይል): Enter your email address (optional)
3. **Subcity** (ክፍለ ከተማ): Enter your subcity
4. **Woreda** (ወረዳ): Enter your woreda
5. **House Number** (የቤት ቁጥር): Enter your house number

### Step 5: Select Target Official
**Screenshot 2.5: Select Official Section**
```
[SCREENSHOT: Dropdown showing list of officials]
- Shows: List of available officials to file complaint against
- Action: Select the official you want to file complaint about
```

**Instructions:**
1. Scroll down to find **"Select Official"** (አመራር ይምረጡ) dropdown
2. Click on the dropdown to see the list of officials
3. Select the official you want to file the complaint against
4. **Important:** You must select an official to proceed

### Step 6: Fill Out Complaint Details
**Screenshot 2.6: Complaint Content Section**
```
[SCREENSHOT: Form showing complaint details fields]
- Shows: Main complaint content, requested solution fields
- Fields:
  * Complaint Main Content (የቅሬታ ዋና ይዘት)

**Instructions:**
1. **Complaint Main Content** (የቅሬታ ዋና ይዘት):

   - Be specific about what happened
   - Include dates and locations if relevant
2. **Requested Solution** (የተጠየቀ መፍትሔ):
   - Describe what solution or action you are requesting
   - Be clear and specific

### Step 7: Additional Information (Optional)
**Screenshot 2.7: Additional Information Section**
```
[SCREENSHOT: Form showing optional fields]
- Shows: Submission Institution, Previous Response fields
- Fields:
  * Submission Institution (የመላኪያ ተቋም) - Optional
  * Previous Written Response (የቀድሞ የተጻፈ ምላሽ) - Optional
```

**Instructions:**
1. **Submission Institution** (የመላኪያ ተቋም): If filing on behalf of an institution, enter the name
2. **Previous Written Response** (የቀድሞ የተጻፈ ምላሽ): If you received a previous response, describe it here

```
[SCREENSHOT: Signature input field with verification]

### Step 9: Submit the Complaint
**Screenshot 2.9: Submit Button**
```
[SCREENSHOT: Form showing Submit button at the bottom]

**Instructions:**

2. Make sure all required fields are filled

### Step 10: Save Your Unique Code
**Screenshot 2.10: Success Modal with Unique Code**
```
[SCREENSHOT: Success modal showing unique code and ticket number]

5. Click **"Close"** (ዝጋ) to close the modal

## 5. Troubleshooting / ችግሮችን መፍታት

### Problem: Can't Submit Complaint
**Solution:**
- Make sure all required fields are filled (marked with *)
- Verify that you selected a target official
- Make sure you verified your signature
- Check your internet connection

### Problem: Can't Find My Unique Code
**Solution:**
- Check your email (if you provided one)
- Contact the office directly with your ticket number
- If you remember the date, they can help locate your submission

### Problem: Can't Select Date/Time
**Solution:**
- Make sure you're selecting a future date
- Check that the time slots are available
- Try selecting a different date or time

### Problem: Form Not Loading
**Solution:**
- Refresh the page (F5 or Ctrl+R)
- Clear your browser cache
- Try a different browser
- Check your internet connection

### Problem: Signature Not Working
**Solution:**
- Try drawing your signature again
- Make sure you click "Verify Signature" after drawing
- Try using a mouse instead of touchpad (or vice versa)
- Clear the signature and try again

### Problem: Can't Follow Up
**Solution:**
- Double-check your Unique Code (8 characters: 4 letters + 4 numbers)
- Make sure you're entering it correctly (case-sensitive)
- Try copying and pasting the code instead of typing
- Contact support if the code still doesn't work

## Important Notes / አስፈላጊ ማስታወሻዎች

1. **Save Your Codes**: Always save your Unique Code and Ticket Number immediately after submission
2. **Required Fields**: Fields marked with * are required
3. **Ethiopian Calendar**: Appointments use Ethiopian calendar dates
4. **Business Hours**: Appointments are available during business hours (3:00 AM - 11:00 AM Ethiopian time)
5. **Response Time**: Officials typically respond within 5-7 business days
6. **Language**: You can switch between English and Amharic using the language selector

## Quick Reference / ፈጣን ማጣቀሻ

### Complaint Unique Code Format:
- 8 characters: 4 letters (A-Z) + 4 numbers (0-9)
- Example: ABCD1234, EFGH5678

### Appointment Unique Code Format:
- 8 characters: 4 letters (A-Z) + 4 numbers (0-9)
- Example: IJKL9012, MNOP3456

### Ticket Number Format:
- Format: GRV-YYYY-XXX
- Example: GRV-2024-123
- YYYY = Year, XXX = Random 3-digit number
